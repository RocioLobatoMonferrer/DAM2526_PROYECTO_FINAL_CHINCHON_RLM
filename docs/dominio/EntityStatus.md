# ENTITYSTATUS

## FUNCIÓN

Enum que indica el estado que se encuentra la entidad.

- INSIDE: La entidad sigue dentro de la partida.
- OUT: La entidad ha sido eliminada de la partida.

## ATRIBUTOS

- Atributos constantes del estado de la entidad:

```java
public enum EntityStatus {
	INSIDE, OUT
}
```
